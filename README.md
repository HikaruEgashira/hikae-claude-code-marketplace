### Hikae Marketplace

#### Skill Chain

| Skill | When to use |
|-------|-------------|
| `assign` | When context is missing |
| `commit-push-pr-flow` | After task completion |
| `review-flow` | After PR creation |

#### Installation

```bash
claude plugin marketplace add HikaruEgashira/hikae-claude-code-marketplace
claude plugin install wf
```
